Цей проєкт це Асистент з документації коду.
Що він робить:

- Користувач вставляє проєкт (github repo або самі файли)
- Бекенд це все дробить на чанки створює embeddings і зберігає в базу даних (Postgres + pgvector)
- Далі користувач пише якесь запитання в чаті ( фронтенд ) бекенд створює embedding для цього запитання і робить пошук по базі даних щоб знайти найбільш релевантні чанки
- передає їх разом із запитом у GPT
- А Ai вже повертає відповідь користувачу

Стек:

- Фронтенд: React(19.2.7) + React Router(8.2.0) + TailwindCSS(4.3.2)
- Бекенд: NestJS(11.0.1) + Prisma(7.8.0) + Postgres(18.0) + pgvector(0.8.5) + OpenAI API (6.46.0)
